# FLX AI Hub

This is a shared storage space — on a site called GitHub — that holds the AI literacy tools built by Finger Lakes Community College's **FLX AI Hub**. If you're new to GitHub or to websites in general, this page explains what's here and how to find what you need, in plain language.

## A quick word on how this is organized

Everything in here lives inside what's called a **repository** (people often shorten it to "repo"). Think of a repository as one big shared folder online. Inside it, things are split into smaller **folders**, the same way you might organize files on your own computer into separate folders for separate projects.

Right now, this repository contains two separate, unrelated tools, each in its own folder:

| Folder | What it is |
|---|---|
| [`ai-challenge/`](ai-challenge/) | A 12-part interactive website that teaches people the basics of AI through short, hands-on challenges. |
| [`engagement-guide/`](engagement-guide/) | A tool instructors use to quickly build a short, clear statement describing how students may (or may not) use AI on a given assignment. |

You generally only need to work with **one** of these folders at a time, depending on which tool you're trying to set up. Each folder is self-contained — meaning everything that particular tool needs lives inside it.

## Getting started

1. **If you want a copy of everything on your own computer:** go to this repository's page, look for the green **Code** button near the top, click it, and choose **Download ZIP**. This downloads both folders above (and anything else in this repository) at once, as a single zipped file you can unzip on your computer.
2. **Open the folder for the tool you want to set up:**
   - For the AI Challenge website, open the `ai-challenge` folder and follow the step-by-step instructions in [`ai-challenge/AI-Challenge.md`](ai-challenge/AI-Challenge.md).
   - For the AI Engagement Guide Generator, open the `engagement-guide` folder and follow the step-by-step instructions in [`engagement-guide/AI-Engagement-Guide.md`](engagement-guide/AI-Engagement-Guide.md).

Each of those guides assumes no prior experience with websites or code, and walks you through everything from there — including, in both cases, an option to run the tool entirely on your own computer with no technical setup, as well as instructions for putting it online if you'd like to share a working link with other people.
